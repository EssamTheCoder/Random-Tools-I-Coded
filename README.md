# Random Tools I Coded
This is a repository of a bunch of Random Tools & Websites I (amongst others) have coded.

## End Goals
These tools should aim to:
- Solve _at least_ one (preferably niche) issue,
- Be secure,
- Be as compact and optimised as possible,
- Look as good as possible,
- Be readable,
- Follow the Style Guide.
These are not requirements, but end goals, the project you make _should_ end development acheiving one of if not most of these goals.

## How to Contribute
Feel free to create any tool you want, just ensure it is in the correct directory (e.g. python projects go in the python repository).
You can make a tool or update or fix one up
There is a Style Guide, your project should aim to follow it by end of development.
As of now, it is very minor, it shouldn't be hard to implement but at the end of the day, it is **optional**
